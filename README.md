Descripcion
Este repositorio contiene el desarrollo del Proyecto 1, que integra datasets abiertos de Costa Rica (INEC, BCCR, Incopesca, UCR).  
El objetivo es analizar el impacto de la inflación en el año 2022 sobre distintos sectores productivos y la percepción ciudadana.

 Sector pesquero en contexto inflacionario  
     Producción Nacional Pesquera (Incopesca)  
     Microempresas (INEC)  
     Inflación (BCCR)  


Estructura del repositorio
   /data_raw  datos originales descargados  
   /data_clean    datos procesados y transformados  
   /notebooks    notebook principal en Google Colab  
    /src  
 /output  resultados exportados 
 /docs  documentación
